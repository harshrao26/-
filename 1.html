HTML Notes

Syntax Help

Fundamentals

  • Any HTML document must start with document declaration <!DOCTYPE html>.
  • HTML documents begin with <html> and end with </html>.
  • Headings are defined with <h1> to <h6>.
  • Paragraphs are defined with <p> tag.
  • Links are defined with <a> tag.

Example:

<a href="https://onecompiler.com/html">HTML online compiler</a>

Images are defined in <img> tag.

Buttons are defined in <button></button>.

Lists are defined in <ul> (unordered) and <ol> (ordered).

List items are defined in <li>.

HTML Elements and Attributes

  • An HTML element includes a start and an end tag.
  • Content between the tags is called HTML element content.
  • Use meaningful tag names.
  • Do not forget ending tags.
  • Empty elements = tags with no content.
  • Attributes provide extra information about elements.

Example:

<a href="https://onecompiler.com/html">HTML online compiler</a>

CSS

CSS describes how HTML elements will look on the web page like color, font-size, background, spacing, etc.

Example CSS:

body {
    padding: 25px;
}

.title {
    color: #228B22;
    font-family: Candara;
}

HTML Tables

  • HTML tables are defined in <table> tag.
  • Table row is defined in <tr>.
  • Table header is defined in <th>.
  • Table data is defined in <td>.
  • Table caption is defined in <caption>.

HTML–JavaScript

  • JavaScript is used to make HTML pages interactive.
  • <script> is the tag used to write or load JavaScript.
  • You can reference external scripts as well.

Example:

<script src="script.js"></script>

Example Question: Ordered List

Below is the HTML code for the ordered list “Festivals of India”: 

<!DOCTYPE html>
<html>
<head>
    <title>Festivals</title>
</head>
<body>

    <h2>FESTIVALS OF INDIA</h2>
    <ol>
        <li>HOLI</li>
        <li>DIWALI</li>
    </ol>

</body>
</html>
